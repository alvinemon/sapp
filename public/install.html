Install 2hotatl on Android

Play Protect often blocks remote-control apps that are not from the Play Store. This is normal for sideloaded accessibility tools.

Download APK
Works remotely — phone on WiFi or mobile data, website on any connection. No USB, no same network.
“App blocked to protect your device”
Google flags any app that uses Accessibility + internet for remote control. 2hotatl is signed and open — you install it for your own phones only.

Method 1 — Install anyway (most phones)

  1. Download the APK above (use Chrome or Files).
  2. When Play Protect warns, tap More details.
  3. Tap Install anyway or Install without scanning.
  4. Open 2hotatl → enter name + email → turn on Watch Sync in Accessibility.
  5. Open 2hotatl.com on any computer — your phone appears automatically.

Method 2 — Pause Play Protect (if Method 1 is hidden)

  1. Open Google Play Store → profile icon → Play Protect.
  2. Tap the gear ⚙ → turn off Scan apps with Play Protect.
  3. Install the APK, then turn scanning back on.

Method 3 — USB install (always works)

  1. Enable Developer options → USB debugging on the phone.
  2. On a computer: adb install 2hotatl.apk
Why Accessibility? Android only allows remote taps through the official Accessibility API. No screen video is sent — only button positions and text.
